Introduction to Charts

Scatter Plot

A scatter plot is a type of plot that displays values for two variables as points on a Cartesian plane. It is useful for identifying relationships between the variables.

Bar Chart

A bar chart is a chart that presents categorical data with rectangular bars with lengths proportional to the values that they represent. It is commonly used to compare data across categories.

Line Chart

A line chart is a type of chart that displays information as a series of data points connected by straight line segments. It is often used to visualize trends over time.

Pie Chart

A pie chart is a circular statistical graphic that is divided into slices to illustrate numerical proportions.

Donut Chart

A donut chart is similar to a pie chart, but with a hole in the center. It is useful for showing the relationship of parts to a whole.

Bubble Chart

A bubble chart is a type of chart that displays three dimensions of data: x, y, and z (size of the bubble). It is useful for visualizing relationships between numeric variables.
